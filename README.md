# 🎮 Projeto 2: Formulário de Venda de Consoles

Este repositório contém o **Projeto 2** do curso, que consiste em um formulário interativo de venda de videogames (PlayStation, Xbox, Nintendo) focado na comunidade de Maranguape - CE. O projeto foi desenvolvido utilizando as tecnologias base da web para criar uma interface moderna e responsiva (estilo "Dark Mode/Gamer").

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura e semântica do formulário (inputs, selects, radios, checkboxes).
* **CSS3:** Estilização visual, uso de variáveis de cores, responsividade com Flexbox e Media Queries.

## 📁 Estrutura de Arquivos

O projeto é composto por apenas dois arquivos principais que devem estar na mesma pasta:

* `index.html`: Contém toda a estrutura e conteúdo do formulário.
* `style.css`: Contém todo o estilo e aparência da página.

## 🚀 Como Executar o Projeto

Não é necessário instalar nenhum programa complexo ou servidor local para visualizar este projeto. Siga os passos abaixo:

1. **Baixe ou extraia os arquivos:**
   Se você baixou os arquivos compactados (.zip), extraia-os primeiro. Certifique-se de que os arquivos `index.html` e `style.css` permaneçam juntos dentro da mesma pasta extraída (por exemplo, `Projeto2_VendaConsoles`).

2. **Abra o arquivo HTML:**
   Dê um duplo clique no arquivo `index.html` para abri-lo diretamente, ou utilize a extensão 'Live Server' no VS Code para uma visualização com atualização em tempo real.

3. **Visualize no Navegador:**
   O arquivo será aberto automaticamente no seu navegador padrão (Google Chrome, Microsoft Edge, Firefox, Safari, etc.). O formulário já estará pronto para interação!

## 📱 Testando a Responsividade

Para testar como o site se comporta em telas de celular:

1. Com a página aberta no navegador (recomenda-se o Google Chrome), aperte a tecla `F12` do seu teclado para abrir as Ferramentas de Desenvolvedor.
2. Clique no ícone 'Toggle device toolbar' (um celular ao lado de um tablet) localizado no canto superior do painel de desenvolvedor, ou pressione `Ctrl + Shift + M`.
3. Escolha o tamanho de tela desejado na barra superior para ver como os campos se adaptam perfeitamente.
